# analise-evasao-ufcg
# Análise de Evasão - UFCG

Este repositório contém a análise estatística da evasão no curso de Ciência da Computação da UFCG, comparando os currículos de 1999 e 2017.

## Estrutura

- dados/: tabelas de evasão por período
- scripts/: scripts em R para geração de gráficos
- outputs/: gráficos gerados

## Objetivo

Analisar o impacto da reforma curricular na evasão estudantil.
